## Organizando o projeto Concessionária SCN

### O software consiste em um sistema .jar criado utilizando como ide o NetBeans e banco de dados o MySql. Para começar a utilizar o projeto é essencial que você tenha uma conta no github, pois assim fica mais fácil saber sempre que tiver uma atividade/atualização no projeto.
### Bom, para começar você baixou o arquivo rar que se encontra em (SOFTWARE_SCN.rar), depois de baixar o arquivo extraia o mesmo e logo em seguida abra, como se trata de um arquivo .jar não é necessário instalar nada. Você abriu o arquivo é se tudo deu certo tem uma tela de login aberta, como você já sabe a função de Registrar Usuário não tá implementada então somente clique no botão 'Entrar', e agora você tá na tela principal, onde você pode cadastrar clientes, veiculos, funcionários, ver os veículos disponíveis e etc...
